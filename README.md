### [商汤科技](https://www.sensetime.com/)

[商汤科技 百度百科](https://baike.baidu.com/item/%E5%8C%97%E4%BA%AC%E5%B8%82%E5%95%86%E6%B1%A4%E7%A7%91%E6%8A%80%E5%BC%80%E5%8F%91%E6%9C%89%E9%99%90%E5%85%AC%E5%8F%B8/16591749)
